== Description ==

WooCommerce YELLOWBOX Shipping Plugin 

Allows your customers to choose the YELLOWBOX quick and convenient
shipping service and define their preferred pickup point in Israel. 



== Installation ==

1. Upload the wp-store- WooCommerce Yellow Plugins

2. Activate the plugin through the ‘Plugins’ menu in WordPress

3. Go to WooCommerce Setting > Shipping

4. Choose Israel > Edit

5. Choose Add Shipping Method

6. Choose Yellow Shipping > and Save with Add Shipping Method

7. Enable Yellow Shipping + Choose EDIT to manage (Cost + Text ):

8. You can control the text view in your site checkout :
a. Shipping Yellowbox Text
b. Choose Button Text
c. Shipping Cost

9. Save Change.
